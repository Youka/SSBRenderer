Font spacing

Addition of space after characters (even the last one) or lines.

RTL (direction)

Geometries placed from right to left. Text rendering depends on internal font renderer RTL calculation. With font space, normal left-to-right text rendering will be performed.

Deformation

Deformation occurs on the already aligned geometry. Before deformation, the geometry path will be splitted to moves + tiny line segments.

Blurring

Box blur kernel, conducted to a general convolution filter (accelerated by MT+SSE2).

3D rotation

Transformations will be multiplied to a 2x3 matrix, no store for depth or perspective. Rotation on x and y axis works just once because of geometry points default depth is 0.

Text spaces

Tabulators will be converted to 4 spaces. An empty last line will be ignored.

Add a new tag

SSBData.hpp:
Add a new tag value store class (see others classes, inheriting from SSBTag). This class transports parsed data to renderer.

SSBParser.cpp:
Parse tag from text in function 'parse_tags' (see other tag parsing). Tag data are collected here.

RenderState.hpp:
Add tag value in native format for renderer to RenderState structure and implent his insertion to RenderState from store instance in function 'eval_tag' (note: normal insertion AND insertion with interpolated value on animation).

Renderer.cpp:
Whereever your tag belongs to, implent it here on the right place to influence rendering.
