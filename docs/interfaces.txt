Avisynth
---------
SSBRenderer.dll is as C Plugin loadable.
Function "SSBRenderer" will be registered.
	clip = SSBRenderer(clip, string, bool)
The string is a SSB script filename. The bool enables warnings on parsing errors (on by default).

########################################################

VirtualDub
----------
Just load SSBRenderer.dll as new filter and add it to your filter list.
A dialog appears to configure the filter.

########################################################

Aegisub
-------
Copy SSBRenderer.dll into the csri folder of Aegisub and set the renderer in Aegisub's options to CSRI/SSBRenderer.
